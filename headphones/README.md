This project is a fully responsive static webpage built from scratch using only HTML and CSS. It is based on a Figma design provided by ALX, and the goal is to reproduce the layout as accurately as possible while following accessibility and responsive design principles.

Figma Design

To complete this project, create a Figma account, open the provided project file, and duplicate it to your drafts. If the main link is not accessible, use the backup link provided in the project instructions.

Font Requirements

If your computer does not have the fonts used in the design, you must download and install:
Source Sans Pro
Spin Cycle OT

Some measurements in the Figma design use decimal values. You are free to round them.

General Requirements

No CSS frameworks are allowed.
No JavaScript is allowed.
The webpage must match the Figma design as closely as possible.
Use semantic HTML, clean CSS, accessibility best practices, and proper responsive design.

Responsive Design Rules

The webpage must switch to the mobile layout at a screen width of 480px or less.
Links must use color #FF6565 when hovered or active.
Buttons must reduce opacity to 0.9 when hovered or active.
The maximum width of the page content should be 1000px, centered on the page.

Project Description

This project demonstrates the ability to translate a Figma UI design into functional HTML and CSS without external libraries. It includes responsive design, accessibility considerations, and pixel-accurate layout reproduction.

Learning Objectives

Understand and interpret Figma design files.
Build layouts using pure HTML and CSS.
Use media queries to implement responsive design.
Apply accessibility standards.
Write clean and organized code.

How to Run the Project

Clone the repository.
Navigate to the headphones directory.
Open index.html in a browser.
